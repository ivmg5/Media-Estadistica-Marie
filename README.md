# Proyecto Media Estadística en MARIE

## Descripción

Este proyecto consiste en un programa desarrollado en el lenguaje ensamblador de MARIE que calcula la media estadística de 10 datos introducidos y muestra el resultado en el puerto de salida. La idea principal es reflexionar sobre el impacto que la transformación digital ha tenido en la industria automotriz, pasando de sistemas mecánico-neumáticos a sistemas electrónico-digitales.

### Contexto

Con este proyecto se pudo comprender cómo la transformación digital ha revolucionado la industria automotriz. Los vehículos actuales no son simplemente máquinas mecánicas, sino sistemas ciberfísicos complejos que integran mecánica, electrónica e informática.

Las operaciones básicas de gestión de datos realizadas en MARIE, como la captura y almacenamiento de datos de sensores y el cálculo de la media estadística, son fundamentales. Estos algoritmos, aplicados en el contexto de un vehículo en movimiento, muestran la importancia de monitorear y ajustar múltiples variables y condiciones en tiempo real. Esto se traduce en una mayor eficiencia y seguridad en la conducción.

Este proyecto subraya la idea de que las disciplinas están convergiendo y se presenta como una oportunidad para remodelar y mejorar la intersección entre la tecnología y la vida cotidiana.

## Código

El programa está estructurado de la siguiente manera:

1. **Entrada de Datos**: Se utiliza un ciclo para introducir los 10 datos.
2. **Cálculo del Promedio**: Una vez introducidos los datos, se realiza el cálculo del promedio mediante un proceso de sustracción iterativa.
3. **Resultado**: El promedio calculado se muestra en el puerto de salida.
